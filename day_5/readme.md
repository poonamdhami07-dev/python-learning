# Python Loops and String Practice

This project contains beginner Python programs for practicing loops, conditions, user input, and string operations.

## Features

### 1. Number Series
Prints numbers from 1 to N.

### 2. Even Number Series
Prints all even numbers within a given range.

### 3. Odd Number Series
Prints all odd numbers within a given range.

### 4. Specific Number Series
Prints numbers with a difference of 3 using the `range()` function.

Example:
```text
1
4
7
10
13
```

### 5. Name Analysis
- Accepts a user's first name.
- Validates that the input contains only letters.
- Prints each character of the name.
- Checks whether the name starts with a vowel or a consonant.

## Concepts Used

- Python `for` loops
- `range()` function
- Conditional statements (`if-else`)
- User input (`input()`)
- String methods (`lower()`, `isalpha()`)
- String iteration

## How to Run

1. Open a terminal in the project folder.
2. Run:

```bash
python loops.py
```
