## What I Learned

### Type Casting

Converted user input from strings to floating-point numbers and performed addition.

```python
a = input("Enter a : ")
b = input("Enter b : ")

p = float(a) + float(b)
print("The sum of a and b is:", p)
````

### User Input

Collected and displayed user information.

```python
name = input("Enter your name: ")
phone_no = input("Enter your phone no.: ")
age = input("Enter your age: ")
```

### Multi-line Strings and Loops

Created a multi-line string and printed each character using a for loop.

```python
i = '''HI
My name is Poonam.
I am CSE student.'''

for character in i:
    print(character)
```

### String Slicing

Practiced different slicing techniques.

```python
a = "Poonam Dhami"

print(a[0:7])
print(a[7:12])
print(a[0:12:2])
print(a[-11:-2])
```
