# Day 1 - Python Basics

## Overview

This project contains my Day 1 Python practice exercises as I begin my programming journey.

I learned:

* Printing text using `print()`
* Taking user input with `input()`
* Converting input to integers using `int()`
* Using comparison operators (`<`, `>`, `==`)
* Applying conditional statements (`if`, `elif`, `else`)
* Building a simple calculator for two numbers


## Program Features

### Welcome Message

Displays a message about my first day of learning Python.

```python
print("Hi! \"This is my first day of learning python on 3 june,2026.\"")
```

### Number Comparison

Takes a number from the user and checks whether it is:

* Greater than 10
* Equal to 10
* Less than 10

Example:

```text
Write a number: 15
a is greater than 10
```

### Simple Calculator

Accepts two numbers from the user and performs:

* Addition
* Subtraction
* Multiplication
* Division

Example:

```text
Enter the first number: 20
Enter the second number: 5

The sum of the two numbers is: 25
The difference of the two numbers is: 15
The multiple of the two numbers is: 100
The division of the two numbers is: 4.0
```
